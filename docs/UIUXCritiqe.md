# Catatan Kritik UI/UX

Terlalu banyak elemen pada layar awal.
Dashboard langsung menampilkan banyak kartu, menu, progress, quest, dan map sekaligus. Untuk anak SMP, ini berpotensi bikin bingung dan cepat lelah secara visual.
Hierarchy belum tegas.
Beberapa elemen sama-sama terlihat penting, padahal seharusnya ada satu fokus utama. Saat ini CTA, statistik, quest, dan progress saling berebut perhatian.
Bagian atas terlalu padat.
Header, greeting, XP bar, stardust, dan tombol utama muncul bersamaan. Ini membuat area paling penting justru terasa sesak.
Sidebar cukup berat untuk user baru.
Menu di kiri terlalu banyak sejak awal. Untuk pengguna SMP, navigasi sebaiknya dibuat lebih sederhana atau bertahap.
Informasi gamification bagus, tapi kebanyakan sekaligus.
Elemen seperti win rate, top score, streak, quest harian, dan leaderboard memang menarik, tetapi jika tampil bersamaan, efeknya berubah jadi overload.
Kontras fokus belum optimal.
Karena banyak blok dengan warna kuat, mata user tidak langsung tahu mana yang harus diklik dulu. Seharusnya hanya satu atau dua elemen yang benar-benar dominan.
Ruang kosong masih ada, tapi belum dipakai untuk mengurangi beban.
Layout terlihat lebar, tetapi isi di dalamnya tetap padat. Jadi ruangnya belum benar-benar membantu memberi napas visual.
Cocok untuk nuansa game, tetapi perlu disederhanakan untuk pelajar SMP.
Visualnya sudah menarik, tapi untuk target usia ini, pendekatan yang lebih ringan dan lebih guided akan lebih efektif.

Kesimpulan evaluasi:
Desain ini menarik dan punya identitas kuat, tetapi untuk anak SMP masih terasa ramai dan agak overwhelming. Secara UX, akan lebih baik jika layar awal dipusatkan pada satu aksi utama, lalu elemen pendukung dibuat lebih ringkas atau bertahap.
